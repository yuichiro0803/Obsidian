# 📁 Projects

Salesforce 導入支援プロジェクトにおける知識・疑問・アプローチを整理するフォルダ。

---

## フォルダ構成

```
📁 Projects/
├── 🐛 Issues/      ← プロジェクト上の疑問点と、それに対するアプローチをセットで管理
├── 📖 Docs/        ← 調査・学習した知識・手法のリファレンスドキュメント
└── 📝 Templates/   ← アプローチの成果物として作成したテンプレート類
```
