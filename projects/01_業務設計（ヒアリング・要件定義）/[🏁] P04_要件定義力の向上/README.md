# P04｜要件定義力の向上 プロジェクト

## 1. プロジェクト概要
顧客の潜在的なニーズを引き出し、技術的に矛盾のない「要件定義書」を迅速かつ正確に作成する能力を高めるための自己研鑽・ナレッジ集約プロジェクト。

## 2. フォルダ構成
- `01_コンセプト・目標定義.md`: 目指すべきレベルと現状の課題
- `02_メソッド・フレームワーク.md`: 体系化された要件定義手法（ユーザーストーリー、非機能要件など）
- `03_タスク・自己評価.md`: スキルアップのためのアクションプランと振り返り
