## 1. 処理インデックス一覧

> **備考**: SmartDBからのステータス同期、ERPからのデータ取得・結果連携に関わるすべてのAPIエンドポイントと処理仕様を明記します。フェーズ（E-01〜E-03）はシーケンス図に対応しています。

| 処理ID | 処理名 (APIクラス) / **URL** | 対象オブジェクト | トリガー / **Method** | 状態遷移 (対象項目) | エラー発生条件 | エラー内容 | ハンドリング方法 |
| :--- | :--- | :--- | :--- | :--- | :--- | :--- | :--- |
| **E-01-A** | 請求先データ取得<br>`BillingInfoAcquisition...`<br>`/BillingInfoAcquisition/*` | 請求先<br>(`BillingInfo__c`) | API (GET) | `確定済`<br>↓<br>`連携中` | CSV生成中・後のUpdate時の例外 | 500 error | ロールバック、メール通知 |
| **E-01-B** | 請求書データ取得<br>`BillingDataAcquisition...`<br>`/BillingDataAcquisition/*` | 請求書明細・ヘッダ | API (GET) | `確定済`<br>↓<br>`連携中` | 同上（※名寄せロジック含む） | 500 error | 同上 |
| **E-02-A** | 請求先連携結果受信<br>`BillingInfoApprove...`<br>`/BillingInfoApprove/*` | 請求先 | API (POST) | `連携中`<br>↓<br>`済` or `エラー` | サイト番号不在、State値異常 | 404/400/500 | ロールバック、メール通知 |
| **E-02-B** | 請求書連携結果受信<br>`CompletionFlagUpdate...`<br>`/CompletionFlagUpdate/*` | 請求書情報 | API (POST) | `連携中`<br>↓<br>`済` or `エラー` | 参照番号不在、State値異常 | 404/400/500 | 同上 |
| **E-03-A** | 請求先Webhook受信<br>`BillingInfoStatusUpdate...`<br>`/BillingInfoStatusUpdate/*` | 請求先 | API (POST) | 承認状況・SDBステータスの更新 | JSON形式不正、**日時逆転** | 400/500 | ロールバック、メール通知 |
| **E-03-B** | 請求書Webhook受信<br>`InvoiceHeaderStatusUpdate...`<br>`/InvoiceHeaderStatusUpdate/*` | 請求書情報 | API (POST) | 承認状況・SDBステータスの更新 | 同上 | 400/500 | 同上 |
