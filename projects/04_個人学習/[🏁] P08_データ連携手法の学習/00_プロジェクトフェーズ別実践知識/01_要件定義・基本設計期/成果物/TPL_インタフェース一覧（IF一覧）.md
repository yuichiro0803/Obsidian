## 記入凡例

### 連携方向
- `→`：Salesforce から外部へ送信（Outbound）
- `←`：外部から Salesforce へ受信（Inbound）
- `↔`：双方向

### 連携方式
| 略称 | 説明 |
|:---|:---|
| REST | Apex Callout / External REST API |
| Bulk | Salesforce Bulk API v2（大量データ向け） |
| PE | Platform Events（イベント駆動型） |
| CDC | Change Data Capture（Salesforce側の変更検知） |
| Flow | Salesforce Flow による自動化 |
| MWare | ミドルウェア経由（MuleSoft / ASTERIA 等） |

### ステータス
`要件定義中` → `基本設計中` → `詳細設計中` → `開発中` → `テスト中` → `完了`
