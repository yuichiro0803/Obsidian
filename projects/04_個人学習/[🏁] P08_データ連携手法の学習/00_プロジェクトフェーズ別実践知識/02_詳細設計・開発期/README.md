# README（フォルダ構成）

## 02_詳細設計・開発期

Salesforceデータ連携案件における詳細設計〜開発フェーズの知識・成果物を管理するフォルダです。

---

## フォルダ構成

### 📚 ガイド/
各フェーズの「進め方・考え方」を整理した学習・参照ドキュメント

| ファイル | 内容 |
|:---|:---|
| [[ガイド/00_詳細設計・開発フェーズ実践ガイド\|実践ガイド]] | ゴール・成果物・6ステップの実践手順 |

### 📋 成果物/
実際のプロジェクトで使用できるテンプレート（`TPL_` プレフィックス）

| ファイル | 内容 |
|:---|:---|
| [[成果物/TPL_プログラム詳細設計書\|詳細設計書]] | クラス・メソッド・フロー・エラー設計のテンプレート |
| [[成果物/TPL_コードレビュー記録\|コードレビュー記録]] | 指摘一覧と5カテゴリのチェックリスト |
| [[成果物/TPL_疑問メモ・課題管理表\|疑問メモ]] | 疑問点・課題の管理表 |

---

> **使い方**
> - **ガイド** → 設計・開発開始前に読んで同フェーズの「型」を確認する
> - **成果物** → `TPL_〇〇` を複製してプロジェクト名を付けて使用する
