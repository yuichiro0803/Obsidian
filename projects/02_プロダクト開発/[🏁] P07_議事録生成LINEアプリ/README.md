# P07｜議事録生成LINEアプリ 開発プロジェクト

## 1. プロジェクト概要
LINE上で音声ファイルまたはテキストを入力すると、Google AI Studio (Gemini API) を活用して構造化された議事録を自動生成するアプリケーションの開発。

- `02_技術スタック・構成案.md`: どのように作るか（AI Studio, LINE Messaging APIなど）
- `03_タスク管理.md`: 開発の進捗管理
