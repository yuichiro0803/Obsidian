# 🏫 Enablement｜業務ヒアリング力の強化 施策一覧

> **ゴール**：各メンバーが、スクラム開発の PBI に記載されているユーザーストーリーを元に、業務ヒアリングを自律的に実施できる状態を作る。

---

## 施策マップ

```
ユーザーストーリーを読む
     ↓
「なぜ」「どのように」業務が行われているかをヒアリングする
     ↓
フロー・ジャーニーの形に整理してPBIに残す
     ↓
次回から自分でできる（自走化）
```
